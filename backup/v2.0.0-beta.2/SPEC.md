# Stage Note 仕様書

## 概要

Stage Note は舞台・ミュージカル・ライブなどの観劇を一元管理するWebアプリ。

---

## 開発方針

- 自分専用から開発を開始する
- 将来的に他ユーザーでも利用できる設計を目指す
- 機能追加前に仕様を決定する
- 小さな単位で実装・確認する
- バージョンごとにCHANGELOGを更新する
- Gitでバージョン管理を行う（導入後）

### バージョン管理

セマンティックバージョニングを採用する。

- メジャー：新しい機能カテゴリ
- マイナー：機能追加
- パッチ：バグ修正・改善

---

## 画面構成

- ホーム
- 観劇記録

## 機能一覧

### 1. 観劇記録

#### 項目

- 公演名
- 観劇日
- 公演時間
- 会場
- 座席
- 感想

#### 機能

- 登録
- 一覧表示
- 観劇日・公演時間順表示
- 削除
- （将来）編集

### 2. チケット管理

#### 基本情報

- 公演名
- 公演日
- 公演時間
- 会場

#### チケット情報

- チケット種別
- 当落
- 発券状況
- 購入場所
- チケット代

#### 将来追加

### ステータス候補

- 申込前
- 申込中
- 当選
- 落選
- 支払済
- 発券済
- 観劇済
- キャンセル
- リセール出品中
- リセール購入待ち
- その他

#### データ設計

| 項目 | 保存名 | 型 |
|------|---------|----|
| ID | id | Number |
| 公演名 | title | String |
| 公演日 | date | String |
| 公演時間 | time | String |
| 会場 | theater | String |
| チケット種別 | type | String |
| 当落 | result | String |
| 受取・発券状況 | issue | String |
| 購入場所 | place | String |
| チケット代 | price | Number |
| 備考 | memo | String |
